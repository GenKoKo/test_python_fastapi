## 🚀 快速開始指南

### **最簡單方式（Codespaces）**

1. 點擊 GitHub 倉庫的 "Code" 按鈕
2. 選擇 "Codespaces" 標籤
3. 點擊 "Create codespace on main"
4. 等待自動設置完成
5. 運行 `just dev` 啟動開發服務器

### **本地開發（Just）**

```bash
# 安裝 Just
brew install just  # macOS

# 設置並運行
just start

# 開發模式
just dev
```

### **Docker 開發**

```bash
# 一鍵 Docker 環境
just docker-start
```
